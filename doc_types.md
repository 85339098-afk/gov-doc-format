# 各文种格式速查

> 本表列出各文种的标题格式、行文方向、结尾用语和特殊要求。

---

## 通知（下行文）

| 项目 | 要求 |
|------|------|
| 标题 | 关于……的通知 |
| 发文字号 | 机关代字〔2026〕×号 |
| 主送机关 | 可有多个，用逗号分隔 |
| 结尾 | 特此通知。 |
| 签发人 | 无需 |

## 请示（上行文）

| 项目 | 要求 |
|------|------|
| 标题 | 关于……的请示 |
| 发文字号 | 机关代字〔2026〕×号 |
| 主送机关 | **只能一个** |
| 结尾 | 妥否，请批示 / 以上请示，请予审批 |
| 签发人 | **必须标注** |
| 特殊 | **一文一事** |

## 报告（上行文）

| 项目 | 要求 |
|------|------|
| 标题 | 关于……的报告 |
| 发文字号 | 机关代字〔2026〕×号 |
| 主送机关 | 可有多个 |
| 结尾 | 特此报告。 |
| 签发人 | **必须标注** |
| 特殊 | 可一文多事，但**不得夹带请示事项** |

## 函（平行文）

| 项目 | 要求 |
|------|------|
| 标题 | 关于……的函 |
| 发文字号 | **机关代字函〔2026〕×号**（带"函"字） |
| 主送机关 | 一函一主送 |
| 结尾 | 特此函复 / 此复 / 请研究函复 |
| 签发人 | 无需 |

## 会议纪要

| 项目 | 要求 |
|------|------|
| 标题 | ×××会议纪要 |
| 发文字号 | 一般不标 |
| 主送机关 | 不标 |
| 署名 | **不加盖公章** |
| 特殊 | 首段记录时间、地点、参会人员、主持人 |

## 批复（下行文）

| 项目 | 要求 |
|------|------|
| 标题 | 关于……的批复 |
| 发文字号 | 机关代字〔2026〕×号 |
| 主送机关 | 通常是来文单位 |
| 首段 | 你单位《关于……的请示》收悉。 |
| 结尾 | 此复 / 特此批复 |
| 签发人 | 无需 |

## 通报（下行文）

| 项目 | 要求 |
|------|------|
| 标题 | 关于……的通报 |
| 发文字号 | 机关代字〔2026〕×号 |
| 主送机关 | 可有多个 |
| 结尾 | 特此通报。 |
| 特殊 | 正文含事实陈述与评价 |

## 决定（下行文）

| 项目 | 要求 |
|------|------|
| 标题 | 关于……的决定 |
| 发文字号 | 机关代字〔2026〕×号 |
| 主送机关 | 可有多个 |
| 结尾 | 本决定自发布之日起施行。 |
| 特殊 | 正文常用"决定如下""特作如下决定" |

## 决议（下行文）

| 项目 | 要求 |
|------|------|
| 标题 | 关于……的决议 |
| 发文字号 | 也可不加 |
| 主送机关 | 不标 |
| 结尾 | （无固定，常用"通过"） |
| 特殊 | 经会议讨论通过，正文首段通常注明通过会议和时间 |

## 命令（令）（下行文）

| 项目 | 要求 |
|------|------|
| 标题 | 发文机关 + 命令（令） |
| 发文字号 | 可编令号，如"第×号" |
| 主送机关 | 不标或统称 |
| 结尾 | （无固定） |
| 特殊 | 发文机关标志使用红色小标宋体，"命令"二字用**二号方正小标宋简体** |

## 公报（下行文/公布性）

| 项目 | 要求 |
|------|------|
| 标题 | ×××公报 / 关于……的公报 |
| 发文字号 | 不标 |
| 主送机关 | 不标 |
| 结尾 | （无固定） |
| 特殊 | 公布重要决定或重大事项；期刊式连续编号 |

## 公告（公布性）

| 项目 | 要求 |
|------|------|
| 标题 | ×××公告 / 关于……的公告 |
| 发文字号 | 可不标，或标注"第×号" |
| 主送机关 | 不标 |
| 结尾 | 特此公告。 |
| 特殊 | 向国内外宣布重要事项；无主送机关 |

## 通告（公布性）

| 项目 | 要求 |
|------|------|
| 标题 | 关于……的通告 |
| 发文字号 | 机关代字〔2026〕×号 |
| 主送机关 | 不标 |
| 结尾 | 特此通告。 |
| 特殊 | 在一定范围内公布应当遵守或周知的事项 |

## 意见（可上行/下行/平行）

| 项目 | 要求 |
|------|------|
| 标题 | 关于……的意见 |
| 发文字号 | 机关代字〔2026〕×号 |
| 主送机关 | 可有多个 |
| 结尾 | 以上意见，请结合实际贯彻落实。 |
| 特殊 | **三向文种**：下行意见可直接执行；上行意见需批复；平行意见供参考 |

## 议案（平行文/上行文）

| 项目 | 要求 |
|------|------|
| 标题 | 关于……的议案 |
| 发文字号 | 机关代字〔2026〕×号 |
| 主送机关 | 只能一个（通常为本级人民代表大会或其常委会） |
| 结尾 | 请予审议。 |
| 特殊 | 政府向人大提出，一事一案；需有提案单位署名 |

---

## 速查矩阵

| 文种 | 方向 | 签发人 | 一文一事 | 结尾用语 |
|------|------|--------|---------|---------|
| 通知 | ⬇ 下行 | ✗ | ✗ | 特此通知 |
| 请示 | ⬆ 上行 | ✓ | ✓ | 妥否，请批示 |
| 报告 | ⬆ 上行 | ✓ | ✗ | 特此报告 |
| 函 | ➡ 平行 | ✗ | ✗ | 特此函复 |
| 纪要 | — | ✗ | ✗ | （无固定） |
| 批复 | ⬇ 下行 | ✗ | ✗ | 此复 |
| 通报 | ⬇ 下行 | ✗ | ✗ | 特此通报 |
| 决定 | ⬇ 下行 | ✗ | ✗ | （无固定） |
| **决议** | **⬇ 下行** | **✗** | **✗** | **（无固定）** |
| **命令** | **⬇ 下行** | **✗** | **✗** | **（无固定）** |
| **公报** | **公布** | **✗** | **✗** | **（无固定）** |
| **公告** | **公布** | **✗** | **✗** | **特此公告** |
| **通告** | **公布** | **✗** | **✗** | **特此通告** |
| **意见** | **⬆⬇➡** | **视方向定** | **✗** | **结合实际贯彻落实** |
| **议案** | **➡⬆** | **✗** | **✓** | **请予审议** |

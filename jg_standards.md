# JG 报送规范参考

> 以下为推荐参考规范，请以各单位实际要求为准。

---

## 常用排版要求

| 项目 | 要求 |
|------|------|
| 字体 | 正文仿宋 3 号（16pt），标题小标宋 2 号（22pt） |
| 行距 | 固定值 28pt（推荐）或 29.5pt（严格对标 GB/T 9704-2012） |
| 页边距 | 上 37mm、下 35mm、左 28mm、右 26mm |
| 页码 | 4 号宋体，— N —，单页居右空一字，双页居左空一字 |

## 参考标准

- **GB/T 9704-2012** — 党政机关公文格式（现行有效标准）
- **GB/T 33476.2-2016** — 党政机关电子公文格式规范 第2部分：显现

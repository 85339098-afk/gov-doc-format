# 排版完成后的自查清单

> 每次生成公文后，逐项核对。

---

## □ 一、页面设置

- [ ] 纸张大小：A4（210mm × 297mm）
- [ ] 上边距：37mm
- [ ] 下边距：35mm
- [ ] 左边距：28mm
- [ ] 右边距：26mm
- [ ] 版心：156mm × 225mm

## □ 二、字体字号

- [ ] 公文标题：方正小标宋简体 二号（22pt）居中
- [ ] 一级标题：黑体 三号（16pt）左空二字
- [ ] 二级标题：楷体 三号（16pt）左空二字
- [ ] 三级标题：仿宋 三号（16pt）
- [ ] 正文：仿宋 三号（16pt）
- [ ] 页码：宋体/ Times New Roman 四号（14pt）

## □ 三、排版格式

- [ ] 正文行距：固定值 28 磅（或 28.95 磅）
- [ ] 首行缩进：2 字符（≈32pt）
- [ ] 段前段后：0 磅
- [ ] 全文颜色：纯黑 RGB(0,0,0)
- [ ] 标点：中文全角，段落结束有句号
- [ ] 正文层次序数规范：一/（一）/1./（1）
- [ ] 标题换行：过长时不超过两行，换行顶格书写，保持编号完整
- [ ] 数字字体：阿拉伯数字建议使用 Times New Roman（正文中的数字）

## □ 四、附件说明（如有时）

- [ ] 正文下空一行，左空二字
- [ ] "附件"二字用 3 号仿宋
- [ ] 多个附件用阿拉伯数字编号
- [ ] 附件名称后不加标点
- [ ] 附件与正文的版式一致

## □ 五、页码

- [ ] 奇偶页不同（单页码居右，双页码居左）
- [ ] 首页不显示页码
- [ ] 格式：— N —
- [ ] 空白页不显示页码

## □ 六、落款

- [ ] 发文单位名称与成文日期之间空一行
- [ ] 成文日期右空四字
- [ ] 日期格式：2026年5月25日（阿拉伯数字）

## □ 七、发文字号

- [ ] 格式：机关代字〔2026〕×号
- [ ] 年份用六角括号〔〕
- [ ] 顺序号不编虚位
- [ ] 标题下空一行，居中排列
- [ ] 函的发文字号带"函"字

## □ 八、签发人（上行文）

- [ ] 请示、报告必须标注签发人
- [ ] 与发文字号同一行，签发人居右
- [ ] "签发人"三字用 3 号仿宋
- [ ] 签发人姓名用 3 号楷体

## □ 九、联合行文

- [ ] 联合行文时，发文机关标志并排
- [ ] 各机关署名按顺序排列
- [ ] 加盖各机关公章
- [ ] 成文日期以最后签发机关为准

## □ 十、版记

- [ ] 版记位于公文末页底部
- [ ] 版记与正文之间用分隔线分开
- [ ] 分隔线宽度与版心等宽
- [ ] 抄送机关后加句号
- [ ] 印发日期右空一字
- [ ] 末页为偶数页（JG 报送规范）

## □ 十一、文种专项

- [ ] **请示**：一文一事 ✓ 主送唯一 ✓ 签发人 ✓
- [ ] **报告**：签发人 ✓ 不夹带请示事项 ✓
- [ ] **函**：发文字号带"函"字 ✓
- [ ] **纪要**：不加盖公章 ✓
- [ ] **批复**：首段引述来文 ✓

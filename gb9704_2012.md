# GB/T 9704-2012 关键条款摘录

> **本标准为现行有效标准**。经国家标准化管理委员会官网核实（https://std.samr.gov.cn），
> GB/T 9704-2012 是唯一发布并生效的"党政机关公文格式"国家标准，
> **不存在 GB/T 9704-2022 版本**。请注意甄别网络上的不实信息。

---

## 1. 范围

本标准规定了党政机关公文通用的纸张要求、排版和印制装订要求、公文格式各要素的编排规则，并给出了公文的特定格式。

## 2. 公文用纸幅面尺寸

- **A4 型纸**：210mm × 297mm

## 3. 版面要求

| 项目 | 要求 |
|------|------|
| 版心 | 156mm × 225mm |
| 上白边（天头） | 37mm ± 1mm |
| 下白边（地脚） | 35mm ± 1mm |
| 左白边（订口） | 28mm ± 1mm |
| 右白边（翻口） | 26mm ± 1mm |

## 4. 字体字号

| 要素 | 字体 | 字号 |
|------|------|------|
| 公文标题 | 方正小标宋简体（小标宋体） | 2 号（22pt） |
| 一级标题（一、） | 黑体 | 3 号（16pt） |
| 二级标题（（一）） | 楷体 | 3 号（16pt） |
| 三级标题（1.） | 仿宋 | 3 号（16pt），可加粗 |
| 正文 | 仿宋 | 3 号（16pt） |
| 页码 | 宋体 | 4 号（14pt） |
| "签发人"三字 | 仿宋 | 3 号 |
| 签发人姓名 | 楷体 | 3 号 |

## 5. 行距（重要说明）

GB/T 9704-2012 规定：正文以 3 号字高度加 3 号字高度 7/8 倍的距离为一行的基准。
根据 GB/T 33476.2-2016《党政机关电子公文格式规范 第2部分：显现》附录 A 的计算：

- 一个 3 号字字高 ≈ **5.54mm**
- 3 号字字高的 7/8 ≈ **4.84mm**
- **一行 ≈ 10.39mm**

在 Word 中以"固定值"设置时，换算为约 **29.5pt**（10.39mm ÷ 25.4 × 72dpi）。

> **实际排版建议**：目前多数机关单位使用固定值 **28pt**，
> 这是实践中形成的通用值，可以接受。如需严格对标标准，可使用 29.5pt。
> 不存在所谓"2022 版改为 28.95pt"的说法。

## 6. 页码

- 页码格式："— N —"
- 单页码居右，双页码居左
- 版记页和空白页不编排页码
- 页码字体：4 号半角宋体
- 一字线上距版心下边缘 7mm

## 7. 公文的特定格式

- **信函格式**：发文机关标志使用红色小标宋体，距上页边 30mm
- **命令格式**：发文机关标志使用红色小标宋体，上边缘至版心上边缘 20mm
- **纪要格式**：上边缘至版心上边缘 35mm

## 8. 装订要求

- 左侧装订
- 骑马订或平订
- 订位为两钉，钉锯与书脊间的距离为 3mm～5mm

## 9. 配套标准

本标准的电子公文实现应同时参考：
- **GB/T 33476.2-2016**《党政机关电子公文格式规范 第2部分：显现》
